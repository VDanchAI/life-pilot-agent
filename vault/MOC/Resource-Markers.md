# 🏷️ Маркеры для Resources Hub

*Как автоматически сохранять полезные материалы в ваш второй мозг*

---

## 📝 Простые маркеры

Используйте эти emoji в начале или конце сообщения:

### 🔗 Ссылки и инструменты
- **Что:** веб-сайты, сервисы, приложения
- **Пример:** "Крутой AI-сервис для дизайна 🔗 https://example.com"
- **Куда попадёт:** Resources.md → Инструменты

### 📚 Статьи и материалы
- **Что:** статьи, видео, подкасты, курсы
- **Пример:** "Интересная статья про продуктивность 📚"
- **Куда попадёт:** Resources.md → Статьи и материалы

### 💎 Особо ценные ресурсы
- **Что:** must-read материалы, лучшие практики
- **Пример:** "Гениальная книга по психологии 💎"
- **Куда попадёт:** Resources.md с пометкой "💎 Особо важно"

---

## 🎯 Текстовые маркеры

Если неудобно искать emoji, используйте текст:

- `#ресурс` или `#resource`
- `сохранить в ресурсы`
- `добавить в хаб`
- `полезная ссылка`

---

## 📋 Примеры использования

### Обычное сообщение
```
Нашёл крутой сервис для создания презентаций
https://gamma.app
```
→ Сохранится как обычная заметка

### С маркером 🔗
```
Нашёл крутой сервис для создаций презентаций 🔗
https://gamma.app
```
→ Попадёт в Resources.md → Инструменты → Дизайн

### С текстовым маркером
```
Нашёл крутой сервис для создания презентаций
https://gamma.app
#ресурс
```
→ Также попадёт в Resources Hub

---

## 🔄 Как это работает

1. **Вы отправляете** сообщение с маркером
2. **Система распознаёт** маркер при обработке `/process`
3. **Автоматически добавляет** в соответствующую категорию
4. **Создаёт wiki-ссылки** для связи с другими заметками

## 💡 Советы

- **Используйте маркеры сразу** — так не забудете важную ссылку

- **Комбинируйте маркеры** — 🔗📚 для полезного инструмента с обучением
- **Не бойтесь экспериментировать** — система умная и разберётся
